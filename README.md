# Listado de países con códigos ISO 3466-1 alfa-2 y ISO 3466-1 alfa-3

## Diccionario de datos:

- **alfa-2**: Código ISO 3166-1 de 2 digitos
- **nombre**: Nombre del país
- **alfa-3**: Código ISO 3166-1 de 3 digitos
